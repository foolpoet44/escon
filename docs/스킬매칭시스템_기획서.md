# 로봇솔루션 Task 조직 스킬 매칭 시스템 기획서

## 📋 Executive Summary

본 기획서는 로봇솔루션 Task 조직의 **미션 및 Enabler별 스킬 매칭 시스템**을 제안한다.
조직의 핵심 미션과 3개 주요 Enabler를 기반으로 필요한 스킬을 체계적으로 매핑하고, 
구성원의 역량 개발 로드맵 및 채용/배치 전략을 지원하는 시스템이다.

---

## 1. 시스템 목적

### 1.1 Why - 왜 필요한가?

- **조직 역량 가시화**: 현재 조직이 보유한 스킬과 부족한 스킬 파악
- **전략적 인재 배치**: 미션/Enabler별 필요 스킬과 구성원 스킬 매칭
- **역량 개발 계획 수립**: Gap 분석을 통한 교육/훈련 우선순위 도출
- **채용 전략 지원**: 조직에 필요한 핵심 스킬 기반의 채용 요건 정의

### 1.2 What - 무엇을 제공하는가?

1. **스킬 온톨로지 구축**: 로봇솔루션 도메인의 스킬 분류 체계
2. **미션-Enabler-스킬 매핑**: 조직 전략과 스킬의 연결 구조
3. **구성원 스킬 프로파일링**: 개인별/팀별 스킬 현황 시각화
4. **스킬 갭 분석**: 목표 vs 현재 스킬 수준 분석
5. **추천 시스템**: 프로젝트-인력 매칭, 학습 경로 추천

---

## 2. 조직 미션 및 Enabler 구조

### 2.1 조직 미션 (Mission)

**"유연하게 변화하는 생산환경에서 사용자가 쉽고 빠르게 재구성 가능한 로봇 자동화 솔루션 제공"**

#### 미션 달성을 위한 핵심 요구사항:
- ✅ 생산 라인 유연성 및 대응력 향상
- ✅ 현장 엔지니어 및 작업자 사용 편의성 극대화
- ✅ AI·로봇 기반 스마트 제조 혁신 지원

---

### 2.2 3대 Enabler (핵심 기술 및 실행 기반)

#### **Enabler 1: Flex RPS 기반 모듈화 구조**
- Plug & Play형 Flex RPS
- Low Code 기반 로봇 시퀀스 프로그래밍
- 표준화된 Interface 제공

#### **Enabler 2: 로봇 솔루션 표준 패키지화**
- 로봇·주변장치·툴링의 표준 셋 제공
- One-click Calibration (로봇 보정 자동화)
- 비전 및 장비 간 쉬운 통합 구조

#### **Enabler 3: IRIS 기반 AI 기술 적용**
- 라인 이벤트 실시간 모니터링
- 원격(리모트) 자동 대응
- 생산 시나리오 분석·로그 기반 최적화

## 3. 스킬 온톨로지 설계

### 3.1 스킬 분류 체계 (4-Level Hierarchy)

```
Level 1: 스킬 도메인 (Domain)
  ├─ Level 2: 스킬 카테고리 (Category)
  │   ├─ Level 3: 스킬 클러스터 (Cluster)
  │   │   └─ Level 4: 개별 스킬 (Skill)
```

#### 예시:

```
🤖 로봇 공학 (Robotics)
  ├─ 로봇 제어 (Robot Control)
  │   ├─ 모션 플래닝 (Motion Planning)
  │   │   ├─ 경로 최적화
  │   │   ├─ 충돌 회피
  │   │   └─ 역기구학 계산
  │   └─ 로봇 프로그래밍 (Robot Programming)
  │       ├─ UR Script
  │       ├─ KUKA KRL
  │       └─ ABB RAPID
```

```
┌─────────────────────────────────────────────────────────────┐
│                    스킬 매칭 시스템 UI                        │

└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────┴────────────────────────────────────────┐
│                   Core Services Layer                        │
├──────────────┬──────────────┬──────────────┬────────────────┤
│ 스킬 온톨로지 │  프로파일링  │  매칭 엔진   │  분석 엔진     │
│   관리       │   서비스     │             │                │
└──────────────┴──────────────┴──────────────┴────────────────┘
                     │
┌────────────────────┴────────────────────────────────────────┐
│                      Data Layer                              │
├────────────────┬────────────────┬────────────────────────────┤
│ 스킬 온톨로지  │  구성원 스킬   │   프로젝트/역할            │

└────────────────┴────────────────┴────────────────────────────┘
```

### 6.2 주요 기능 모듈

#### **Module 1: 스킬 온톨로지 관리**
- 스킬 분류 체계 관리 (CRUD)
- 스킬 간 관계 정의 (prerequisite, related)
- 버전 관리 및 히스토리 추적

#### **Module 2: 구성원 스킬 프로파일링**
- 개인별 스킬 등록 및 업데이트
- 자가 평가 + 리더 평가 + 객관적 평가 (프로젝트 실적 기반)
- 스킬 레벨 정의: Beginner → Intermediate → Advanced → Expert

#### **Module 3: 미션/Enabler-스킬 매핑 관리**
- 조직 전략과 스킬 연결
- Enabler별 필요 스킬 정의 및 가중치 설정
- 우선순위 스킬 식별

#### **Module 4: 스킬 갭 분석**
- 조직 전체 스킬 현황 vs 목표 비교
- 팀별/개인별 갭 분석

- 조직 스킬 맵 시각화
- Enabler별 스킬 커버리지 분석

id: string

4. 팀 구성 후, 부족 스킬에 대한 교육 계획 자동 생성

**효과**: 팀 구성 시간 70% 단축, 스킬 미스매치 최소화

**상황**: 신입 엔지니어 A씨의 1년 성장 로드맵 수립

**시스템 활용**:

   - Month 1-3: 로봇 기구학 온라인 강의
   - Month 4-6: Hand-Eye Calibration 실습
   - Month 7-12: VGR 프로젝트 참여

**효과**: 명확한 성장 목표 설정, 학습 동기 부여

### Scenario 3: 조직 스킬 갭 전략 수립

**상황**: 연간 인력 개발 계획 수립

**시스템 활용**:

   - 예: "이상 탐지 알고리즘" 보유자 2명 → 목표 7명
3. 우선순위 기반 교육 예산 배분
4. 외부 채용 vs 내부 육성 의사결정 지원

**효과**: 데이터 기반 전략 수립, 투자 효율성 극대화
