## 📊 구현 개요

### 생성된 파일
1. **`app/lib/comparison.ts`** - 도메인 비교 로직 (Jaccard 유사도, 공통 스킬 추출 등)
2. **`app/components/DomainSelector.tsx`** - 도메인 선택 UI
3. **`app/components/ComparisonChart.tsx`** - Recharts 기반 비교 차트
4. **`app/compare/page.tsx`** - 도메인 비교 페이지

### 주요 기능

#### 1. **도메인 비교 로직** (`comparison.ts`)
- 두 도메인의 스킬 리스트를 비교하여 공통/고유 스킬 분류
- **Jaccard Similarity** 알고리즘을 사용한 유사도 점수 계산
- 스킬 타입별(Knowledge vs Competence) 분포 분석

#### 2. **도메인 선택 UI** (`DomainSelector.tsx`)
- 직관적인 2개의 드롭다운 메뉴
- 'VS' 배지를 통한 대결구도 시각화
- 반응형 디자인 및 스타일링

#### 3. **비교 결과 시각화** (`ComparisonChart.tsx`)
- **Bar Chart**: 전체 스킬 수 및 고유 스킬 수 비교
- **Pie Chart**: 각 도메인의 스킬 유형(Knowledge/Competence) 분포 시각화
- Recharts 라이브러리 활용하여 반응형 차트 구현
- 툴팁 및 범례 제공

#### 4. **비교 페이지** (`compare/page.tsx`)
- 사용자가 두 도메인을 선택하면 실시간으로 분석 결과 표시
- 유사도 점수 카드로 핵심 지표 강조
- 공통 스킬 목록 그리드 뷰 제공
- 데이터가 없을 경우 안내 메시지 표시
